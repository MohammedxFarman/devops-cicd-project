# DevOps CI/CD Project

## Overview
This project demonstrates deploying a Flask application using Docker on AWS EC2.

## Tech Stack
- AWS EC2
- Docker
- GitHub
- Flask

## Workflow
1. Build Docker image locally
2. Push code to GitHub
3. Clone repo on EC2
4. Run container using Docker
5. Access application via public IP

## Output
The application runs on EC2 and is accessible via browser.
